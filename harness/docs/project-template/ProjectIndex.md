# Project Template Index

## 1. 层级定位

Project Template Layer 用于定义通用 Harness Template 在真实项目中实例化后的项目文档结构。

通用 Harness 仓库中使用：

```text
docs/project-template/
```

真实项目实例中使用：

```text
docs/project/
```

| `docs/project-template/prd/` | planned | 项目需求文档模板 |
